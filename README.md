# Modulo__3
Este bloque estuvo dedicado al aprendizaje práctico de Simul8 y a la aplicación de sus variables operativas para el diagnóstico y mejora de sistemas industriales. Desarrollamos una simulación digital a medida que reproduce un esquema de fabricación continua, integrando de manera encadenada los subprocesos de molienda, clarificación, cristalización y el posterior centrifugado.
